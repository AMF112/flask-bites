About Flask Bites

Flask Bites is a functional web application developed to demonstrate the core principles of full-stack development.

Project Goals

The primary objective of this project was to create a clean, responsive ordering interface that handles dynamic data. It focuses on user authentication, database management with PostgreSQL, and a modern UI using Bootstrap 5.

Technical Stack

Python Flask PostgreSQL SQLAlchemy Jinja2

Developer

Built by Adrian Flores, a Computer Science student focusing on Software Engineering and scalable web architectures.
